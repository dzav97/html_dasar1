Halaman 2

Membuat struktur dasar halaman web: Program ini mendefinisikan struktur dasar halaman web dengan tag 'html', 'head', dan 'body'. Tag 'h1' digunakan untuk judul halaman, tag 'p' untuk paragraf, dan tag 'hr' untuk garis horizontal. Konten eksternal: Program ini menggunakan tag "iframe" untuk menyisipkan tiga dokumen HTML eksternal ke dalam halaman web. Iframe memungkinkan Anda untuk menampilkan konten dari situs web lain di dalam halaman web Anda. Membuat tautan: Program ini menambahkan tautan dengan teks "Halaman 2" yang mengarah ke "latihan_1_2.html". Ketika tautan diklik, konten "latihan_1_2.html" akan dimuat di dalam iframe yang memiliki nama "if". Format teks: Program ini menunjukkan cara untuk menerapkan format teks strikethrough (̶t̶e̶k̶s̶) pada paragraf di halaman web.
